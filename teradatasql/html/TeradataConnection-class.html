Arguments

conn

A TeradataConnection object

name

A character string specifying a table name

value

A data frame

row.names

Either TRUE, FALSE, NA or a string. If TRUE, translate row names to a column named "row_names". If FALSE, never translate row names. If NA, translate row names only if they are a character vector.

dbObj

A TeradataConnection object

obj

An R object whose SQL type we want to determine

statement

A character string containing the SQL request text

prefix

A database name or NULL.

temporary

If TRUE, compose a SQL request to create a volatile table.

con

A TeradataConnection object

table

A character string specifying a table name

fields

Either a character vector or a data frame A named character vector: Names are column names, values are types. A data frame: field types are generated using dbDataType().
